The default mesorah is Modern Israeli. If you'd like to change it, just change the              
latin letter that comes before the "#" and the name of the letter.                                          
                                                                                                                                                               
EXAMPLE:                                                                                                                                          
 1511: 'ch', # Qof                                                                                                                               
          ↓                                                                                                                                              
 1511: 'kh', # Qof                                                                                                                               
                                                                                                                                                               
You'll also have to change the final version of the letter            
                                                                                                                                                                                                            #
 1498: 'ch',# Final Chaf                                                                                                                      
 1499: 'ch',# Chaf                                                                                                                               
        ↓                                                                                                                                              
 1498: 'kh',# Final Chaf                                                                                                                      
 1499: 'kh',# Chaf
